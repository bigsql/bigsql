Installation New! Dec 27 - v5.3.0

It's easy to get started with a single-node cluster on CentOS 7. With a few simple commands you'll be up and running with a powerful sandbox in one minute.


1.) From your $HOME directory, as a non-root user, run this command to install our package manager 
python -c "$(curl -fsSL https://s3.amazonaws.com/pgcentral/install.py)"

2.) Change into the $HOME/bigsql directory and install GPDB 
pgc install gpdb5

3.) Start the single node cluster (it will initialize automagically the first time) 
pgc start gpdb5
