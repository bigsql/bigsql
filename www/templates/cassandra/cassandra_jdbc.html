Cassandra JDBC

This Cassandra JDBC Driver implements a robust subset of the JDBC specification as appropriate for use with C*2+ & JDK7+.

Download cassandra-jdbc-2.1-2  

Source Code    Issue Tracker

Features

  • Cassandra 2.x & 3.x compatibility
  • Examples and Lib directories demonstrating usage
  • Cluster topology discovery
  • Round robin connection scheme
  • Primary and Backup DC configuration through JDBC URL

Sample Code

 

import java.sql.Connection;
import java.sql.PreparedStatement;

...

Class.forName("org.bigsql.cassandra2.jdbc.CassandraDriver");
Connection con = DriverManager.getConnection
  ("jdbc:cassandra://host1--host2--host3:9160/keyspace1?primarydc=DC1&backupdc=DC2&consistency=QUORUM");

String query = "UPDATE Test SET a=? WHERE KEY=?";
PreparedStatement statement = con.prepareStatement(query);

statement.setLong(1, 100);
statement.setString(2, "key0");

statement.executeUpdate();

statement.close();
